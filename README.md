# Placement Prediction System

## Project Overview
This project predicts whether a student will be placed and estimates the expected salary using Machine Learning.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithms
- Logistic Regression
- Linear Regression

## Features
- Data Preprocessing
- Placement Prediction
- Salary Prediction
- Data Visualization
- Correlation Heatmap
- Performance Evaluation

## How to Run
1. Open the notebook in Google Colab.
2. Upload the dataset (CSV file).
3. Run all cells.
4. View the predictions and graphs.
